# 🛡️ Defesa Antiaérea Cibernética

## 🎯 Objetivo
Este projeto educacional simula um sistema de defesa antiaérea fictício, com foco em **cibersegurança**.  
A ideia é demonstrar como proteger sensores, radares e comunicações contra ataques digitais, aplicando boas práticas de defesa.

---

## 📂 Estrutura do Projeto
- `arquitetura.md` → descrição da arquitetura do sistema.
- `ameacas.md` → principais ameaças cibernéticas.
- `logs_simulados.json` → eventos fictícios de ataque.
- `resposta_incidentes.md` → playbook de resposta a incidentes.
- `diagramas/` → representações visuais em ASCII.

---

## ⚠️ Principais Ameaças Simuladas
- Ataques **DDoS** contra servidores do centro de comando.  
- **Spoofing de radar** para inserir sinais falsos.  
- **Intrusão em rede** para roubo ou manipulação de dados.  
- **Malware em consoles** dos analistas.  
- **Interceptação de comunicação** entre unidades.
