# Defesa Antiaérea Cibernética


## Objetivo
Projeto educacional que simula um sistema de defesa antiaérea fictício, com foco em **cibersegurança**.  
O objetivo é mostrar como proteger sensores, radares e comunicações contra ataques digitais.

## Estrutura
- `arquitetura.md`: descrição da arquitetura do sistema.
- `ameacas.md`: principais ameaças cibernéticas.
- `logs_simulados.json`: eventos fictícios de ataque.
- `resposta_incidentes.md`: playbook de defesa.
- `diagramas/`: fluxos visuais.

## Aviso
Este projeto é **simulado** e criado apenas para fins de aprendizado.  
Não contém instruções militares reais.

## Diagramas

Este projeto inclui representações visuais em ASCII para facilitar a compreensão:

- `diagramas/arquitetura.txt`: mostra a arquitetura geral do sistema de defesa antiaérea.
- `diagramas/fluxo_ataque_defesa.txt`: mostra o fluxo de ataque e as respostas defensivas.

Esses diagramas ajudam a visualizar como os componentes se conectam e como a cibersegurança atua em cada etapa.
